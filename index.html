数据类型

强语言类型

​ 要求变量的使用要严格符规定，所有变量都必须先定义后使用

弱语言类型

​ 数据类型可以被忽略的语言。它与强类型定义语言相反, 一个变量可以赋不同数据类型的值。

Java的数据类型分为两大类

​ 基本类型（primitive type）
​ 引用类型（reference type）

数据类型 描述 占用字节 取值范围 默认值
int 整型，用于存储整数 4个字节 -2147483648~2147483647 0
byte Java中最小的数据类型 1个字节 -128~127 0
short 短整型 2个字节 -32768~32717 0
long 长整型 8个字节 -2^63~2^63-1 0L
float 浮点型，用于存储带小数点的数字 4个字节 / 0.0f
double 双精度浮点型，用于存储带有小数点的数字 8个字节 / 0.0d
char 字符型，用于存储单个字符 2个字节 0~65535 空
boolean 布尔类型，用于判断真或假 1个字节 仅有两个值，即true、false false

注释

什么是注释

注释标签是使用来在源代码中添加说明/注释，使用该标签注释的内容并不会被执行。

注释的作用

  • 有助于他人阅读代码快速理解

  • 有助于提高开发效率

  • 有助于保持代码一致性

  • 帮助修补程序或快速修复

  • 有助于加快开发过程

  • 有助于提高协作效率

Java注释有三种

  • 单行注释

  • 多行注释

  • 文档注释

Java是什么

2021-12-12
Java学习 / JavaSE / Java是什么

Java概述

什么是Java？

Java是Sun Microsystems于1995年首次发布的一种编程语言和计算平台。编程语言还比较好理解，那么什么是计算平台呢？

计算平台是在电脑中运行应用程序（软件）的环境，包括硬件环境和软件环境。一般系统平台包括一台电脑的硬件体系结构、操作系统、运行时库。

Java是快速，安全和可靠的。从笔记本电脑到数据中心，从游戏机到科学超级计算机，从手机到互联网，Java 无处不在！Java主要分为三个版本

  • JavaSE（J2SE（Java2 Platform Standard Edition，java平台标准版）
  • JavaEE（J2EE（Java 2 Platform，Enterprise Edition，java平台企业版）
  • JavaME（J2ME）（Java 2 Platform Micro Edition，java平台微型版）。
